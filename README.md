# AI Engineering One-Year Study Repository (ai_source)

This repository organizes a 12‑month intensive AI engineering curriculum and contains:
- curriculum/: month-level guides and resource lists
- projects/: week-level code projects (Weeks 1–12 sample included)
- tools/: environment and CI/CD templates
- datasets/: sample data and pointers
- infra/: Docker, k8s, Terraform examples
- portfolio/: project showcase scaffolding
